Admin Control – Receipt Disable / Enable

This admin page can disable the receipt system globally if misuse happens.
Works with a simple Google Apps Script Web App (JSON API).

1) Paste your Web App URL in both this Admin page and the Receipt page.
2) Use Disable to stop all form filling immediately.
Status: —
Apps Script API format (you can deploy in Google Apps Script):
• ?action=get → returns JSON { enabled: true/false, note: "..." }
• ?action=set&enabled=1&token=YOURTOKEN¬e=...
